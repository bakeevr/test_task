# Утилита анализа данных по странам

Утилита для вычисления средних показателей по странам из CSV файлов с выводом результатов в табличном формате.

## Особенности
* Чтение данных из CSV файлов
* Вычисление средних значений по странам
* Вывод результатов в табличном формате
* Гибкая настройка через аргументы командной строки
* Полное покрытие тестами
* Обработка различных ошибок ввода/вывода

## Установка
Клонируйте репозиторий:

`git clone https://github.com/bakeevr/test_task.git`

`cd test_task`

Установите зависимости:
`uv pip install requirements.txt`

## Использование
`python src/main.py --files файл1.csv файл2.csv --report average-название_столбца`

## Тестирование
Запуск всех тестов

`pytest .`

Запуск конкретных тестов

`pytest tests/test_main.py -v`

`pytest tests/test_reader.py -v`
